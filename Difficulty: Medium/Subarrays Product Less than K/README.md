<p><span style="font-family: arial, helvetica, sans-serif; font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><span style="font-family: arial,helvetica,sans-serif;"><strong style="font-family: arial, helvetica, sans-serif; font-size: 14pt;">Input :</strong><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 14pt;"> k = 10, arr[] = [1, 2, 3, 4]

</span></span><strong style="font-family: arial, helvetica, sans-serif; font-size: 14pt;">Explanation:</strong><span style="font-family: arial, helvetica, sans-serif;">

<pre><span style="font-family: arial, helvetica, sans-serif; font-size: 14pt;"><strong>Input:</strong> k = 100, arr[] = [1, 9, 2, 8, 6, 4, 3]
